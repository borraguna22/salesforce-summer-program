 📘 What is Salesforce?

Salesforce is a cloud-based CRM platform used to:
- Manage customer relationships
- Store business data
- Automate business processes
- Build custom business applications

Companies use Salesforce to connect:
- Sales
- Service
- Marketing
- Finance
- Support teams

in one platform.

---

# 🧠 What is CRM?

## CRM = Customer Relationship Management

CRM is a system used to:
- Store customer information
- Track customer interactions
- Manage sales and support
- Improve customer relationships

---

# 🔄 How CRM Fits into Salesforce Platform

Salesforce platform is the complete cloud system.

CRM is one of the main applications built on the Salesforce platform.

---

## Simple Understanding

```text
Salesforce Platform
        ↓
CRM Applications
        ↓
Customer Data + Business Processes
```

---

# 📦 Salesforce Platform Structure

Salesforce platform has different layers.

---

# 🔹 1️⃣ User Interface Layer

This is the front-end users interact with.

Includes:
- Apps
- Tabs
- Dashboards
- Reports
- Record Pages

Example:
- Sales App
- Service App

---

# 🔹 2️⃣ Application Layer

This layer contains business applications.

Examples:
- Sales Cloud
- Service Cloud
- Marketing Cloud

These apps help businesses manage different operations.

---

# 🔹 3️⃣ Logic Layer

Handles automation and business logic.

Includes:
- Flows
- Validation Rules
- Apex Code
- Process Automation

Example:
```text
Automatically send email when lead is created
```

---

# 🔹 4️⃣ Data Layer

Stores business data.

Uses:
- Objects
- Fields
- Records

Example:

| Object | Stores |
|---|---|
| Account | Company data |
| Contact | Customer details |
| Lead | Potential customers |

---

# 🔹 5️⃣ Security Layer

Controls user access and security.

Includes:
- Profiles
- Roles
- Permission Sets
- Sharing Rules

This keeps company data secure.

---

# ☁ Salesforce Cloud Architecture

Salesforce works on cloud infrastructure.

Users access Salesforce using internet browsers.

```text
User
  ↓
Internet
  ↓
Salesforce Cloud Platform
  ↓
Database and Services
```

---

# 🏢 Multi-Tenant Architecture

Salesforce uses multi-tenant architecture.

This means:
- Multiple companies share same infrastructure
- Each company has separate data and security

---

## Simple Understanding

```text
One Salesforce Platform
        ↓
Many Companies Use It Securely
```

---

# 📱 What is an App in Salesforce?

An App is a collection of:
- Tabs
- Objects
- Features
- Tools

used for a specific business purpose.

---

## Example Apps

| App | Purpose |
|---|---|
| Sales App | Sales management |
| Service App | Customer support |
| Marketing App | Marketing campaigns |

---

# 📦 What is an Object?

Objects are database tables used to store data.

---

## Types of Objects

### Standard Objects
Created by Salesforce.

Examples:
- Account
- Contact
- Opportunity

---

### Custom Objects
Created by developers or admins.

Examples:
- Student
- Employee
- Hospital

---

# 🧭 What is Navigation?

Navigation means moving between:
- Apps
- Tabs
- Records
- Pages

inside Salesforce.

Navigation helps users quickly access features and data.
